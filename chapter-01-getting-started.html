Section 1.1 Writing a Simple C++ Program | 编写一个简单的 C++ 程序

每个 C++ 程序包含一个或多个函数, 其中一个必须命名为 main. 操作系统通过调用 main 来运行 C++ 程序. 

// main 函数: 返回给操作系统一个值
int main() {
  return 0;
}

函数的定义包括返回类型, 函数名, 形参列表和函数体.

main 函数的返回类型必须为 int 整数类型. main 函数的返回值用于指示状态: 0 表明成功, 非 0 通常用来指出错误类型.

return 结束函数的执行. 当 return 返回一个值时，该返回值的类型必须与函数的返回类型相容.

1.1.1 Compiling and Executing Our Program | 编译, 运行程序

C++ 源文件常见的后缀命名约定包括: .cc, .cxx, .cpp, .cp, .C. 

# 编译, 执行 C++ 源文件, 获取返回值
$CC program-1.cc          #  $ 为系统提示符, CC 为编译器的名字, 编译器生成可执行文件

# Windows 系统
$.\program-1              # 默认输出 program-1.exe
$echo %ERRORLEVEL%       # 获取返回值

# Unix 和 Unix-like 系统
$./a.out                 # 通常输出 a.out
$echo $?                 # 获取返回值

Windows 系统运行可执行文件可以忽略拓展名 .exe, Unix 和 Unix-like 系统需要使用全名. 

# GCC 编译器
$g++ -o program-1 program-1.cc # -o program1 指定可执行文件的文件名
$g++ -std=c++11 program-1.cc  # -std=c++11 打开 C++11 的支持

# MSVC 编译器
$cl /EHsc program-1.cc        # /EHsc 打开标准异常处理

GCC 编译器的 -Wall 选项和 MSVC 编译器的 /W4 选项可以对有问题的程序结构发出警告.

Section 1.2 A First Look at Input/Output | 初识输入输出

iostream 库包括输入流 istream, 输出流 ostream 等. 标准库定义了 4 个 IO 对象: 标准输入 cin, 标准输出 cout, 标准错误 cerr 和 clog. cerr 输出警告和错误消息, clog 输出程序运行时的一般性信息. 

// 提示用户输入两个整数，输出它们的和
#include <iostream>                                                                                                // #include 指令指出头文件, 一般放在源文件的开始位置
int main() {
  std::cout << "Enter two numbers: " << std::endl;                                                                 // 作用域运算符 ::, 表示使用定义在命名空间中的名字
  int variable_1 = 0, variable_2 = 0;
  std::cin >> variable_1 >> variable_2;
  std::cout << "The sum of " << variable_1 << " and " << variable_2 << " is " << variable_1 + variable_2 << std::endl; // endl 为操纵符的特殊值, 结束当前行, 并将与设备关联的缓冲区中的内容刷到设备
  return 0;
}

输出运算符 <<, 左侧为 ostream 对象, 右侧为要打印的值. 

std::cout << "Enter two numbers: " << std::endl;
// 等价于
(std::cout << "Enter two numbers: ") << std::endl;
// 等价于
std::cout << "Enter two numbers: ";
std::cout << std::endl;

输入运算符 >>, 左侧为 istream 对象, 右侧为要读入的数据. 

std::cin >> variable_1 >> variable_2;
// 等价于
(std::cin >> variable_1) >> variable_2;
// 等价于
std::cin >> variable_1;
std::cin >> variable_2;

Section 1.4 Flow of Control | 控制流

1.4.1 The while Statement | while 语句

// 使用 while 语句求 1 到 10 这 10 个数的和
#include <iostream>
int main() {
  int sum = 0, value = 1;
  while(value < 11) {
    sum += value;
    ++value;
  }
  std::cout << "Sum of 1 to 10 inclusive is " << sum << std::endl;
  return 0;
}

复合赋值运算符 +=, 将右侧运算对象加到左侧运算对象, 结果保存在左侧运算对象. 

sum += value;
// 等价于
sum = sum + value;

前缀递增运算符 ++, 将运算对象的值增加 1. 前缀递减运算符 --, 将运算对象的值减少 1. 

++value;
// 等价于
value = value + 1;

}

1.4.3 Reading an Unknown Number of Inputs | 读取数量不定的输入数据

Windows 系统的文件结束符为 Ctrl + Z, 然后按 Enter 或 Return 键. Unix 和 Unix-like 系统的文件结束符为 Ctrl + D. 

// 读取不定数量的数据并求和
#include <iostream>
int main() {
  int sum = 0, value = 0;
  while(std::cin >> value) { // 遇到文件结束符或无效输入的时候, 条件为假
    sum += value;
  }
  std::cout << "Sum is: " << sum << std::endl;
  return 0;
}

1.4.4 The if Statement | if 语句

// 统计在输入中的每个值连续出现的次数
#include <iostream>
int main() {
  int current_value = 0,  value = 0; // current_value 正在统计的数, value 读入的新值
  if(std::cin >> current_value) {    // 确保确实有数据可以处理
    int count = 1;
    while(std::cin >> value) {
      if(value == current_value) {
        ++count;
      }
      else {
        std::cout << current_value << " occurs " << count << " times." << std::endl;
        current_value = value;
        count = 1;
      }
    }
    // 不要遗忘: 打印最后一个值的个数
    std::cout << current_value << " occurs " << count << " times." << std::endl;
  }
  return 0;
}

C++ 用 = 进行赋值, 用 == 作为相等运算符.

1.5.2 A First Look at Member Functions | 初识成员函数

成员函数是定义为类的一部分的函数, 也称为方法. 

// 检查两个对象是否有相同的 ISBN
#include <iostream>
#include "Sales_item.h"
int main() {
  Sales_item item_1, item_2;
  std::cin >> item_1 >> item_2;
  if(item_1.isbn() == item_2.isbn()) {  
    std::cout << item_1 + item_2 << std::endl;
    return 0;
  }
  else {
    std::cout << "Data must refer to same ISBN" << std::endl;
    return -1;
  }
}

Section 1.6 The Bookstore Program | 书店程序

// 从文件中读取销售记录, 输出册数, 总销售额和平均售价
#include <iostream>
#include "Sales_item.h"
int main() {
  Sales_item total;
  if(std::cin >> total) {
    Sales_item trans;
    while(std::cin >> trans) {
      if(total.isbn() == trans.isbn()) {
        total += trans;
      }
      else {
        std::cout << total << std::endl;
        total = trans;
      }
    }
    std::cout << total << std::endl;
  }
  else {
    std::cout << "No data?!" << std::endl;
    return -1;
  }
  return 0;
}

Chapter Summary | 小结

Defined Terms | 术语表

  • 默认情况下, 读 cin 会刷新 cout, 程序非正常终止时也会刷新 cout.
  • 默认情况下, cerr 的数据不缓冲.
  • 默认情况下, clog 的数据是缓冲的.
  • 条件: 求值结果为真或假的表达式. 通常用值 0 表示假, 用非零值表示真.
